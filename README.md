🚀 AI Job Market Intelligence Platform

An interactive data analytics and machine learning dashboard that analyzes job market data to provide insights into salary trends, skill demand, hiring companies, and industry demand.

This project uses machine learning and data visualization to help understand the AI job market landscape.

📊 Features
💰 AI Salary Prediction

Predicts estimated salary based on company and location using a machine learning model.

🧠 Skill Demand Analysis

Identifies the most in-demand skills from job listings.

🌍 Job Demand by Location

Shows which locations have the highest number of job openings.

🏢 Top Hiring Companies

Highlights companies that are actively hiring the most.

📈 Salary Distribution

Visualizes salary ranges across different jobs.

🔍 Industry Job Demand

Analyzes which industries have the highest hiring demand.

📊 Interactive Dashboard

Built using an interactive dashboard to explore job market insights in real time.
